# Getting Started

To get a local copy up and running follow these simple example steps:


## Setup
- Install code editor
- Install Git bash

## Install
- clone the [project](https://github.com/sj1978/math-magicians.git).
- Run npm install
- Run npm run start to run the project

### Run tests
- Run npm test
